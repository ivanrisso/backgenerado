# Política de Idioma de Artefactos

## Regla global (NO negociable)

Todo archivo generado por cualquier workflow del sistema
DEBE estar escrito íntegramente en **ESPAÑOL**.

Incluye (no exhaustivo):
- Workflows 03, 70, 71 y 72
- Runbooks
- Hotfix descriptions
- Fix prompts
- Implementation plans
- Evidencias QA
- Status.md
- Gates
- Casos de prueba

## Excepciones
- Código fuente
- Identificadores técnicos (variables, clases, endpoints, imports)
- Logs crudos del sistema

⚠️ Si un archivo se genera en otro idioma → el Gate correspondiente **DEBE FALLAR**.
