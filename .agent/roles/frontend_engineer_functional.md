# Rol — Frontend Engineer (Funcional)

## Misión
Implementar cambios funcionales en UI sin romper estabilidad técnica.

## Responsabilidades
- Ajustar flujos
- Corregir validaciones
- Reflejar reglas funcionales reales

## Restricciones
- No refactor general
- No cambios estéticos
- No tocar contratos innecesariamente

## Outputs
- Cambios en vistas / composables
