# SKILL — Allowed Stack Definition

## Rol autorizado
- Arquitecto

---

## Objetivo

Definir el **stack tecnológico permitido** para el sistema.

Todo stack no listado se considera **NO autorizado**.

---

## Output obligatorio

.architecture/allowed_stack.md

---

## Reglas

- Lenguajes permitidos
- Frameworks permitidos
- Bases de datos permitidas
- Infraestructura permitida
- Herramientas CI/CD
