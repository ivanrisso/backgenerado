# SKILL — Architectural Principles Definition

## Rol autorizado
- Arquitecto

⚠️ Ningún otro rol puede ejecutar este skill.

---

## Objetivo

Definir los **principios arquitectónicos rectores** del sistema
que gobiernan todas las decisiones técnicas y funcionales futuras.

Este skill:
- ✔️ Define reglas duras
- ✔️ Justifica decisiones
- ❌ NO diseña implementación
- ❌ NO genera código

---

## Output obligatorio

.architecture/architectural_principles.md

---

## Reglas (NO negociables)

- Cada principio debe ser explícito
- Cada principio debe tener justificación
- Todo lo no definido queda prohibido
