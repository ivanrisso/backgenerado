# SKILL — Hotfix Describer

## Rol autorizado
- Arquitecto de Software Senior

## Objetivo
Documentar cada hotfix técnico sin tocar código.

## Inputs
- Issues clasificados como Hotfix

## Pasos
1. Crear carpeta HF-XXX.
2. Documentar:
   - Error
   - Ubicación
   - Impacto
   - Criterio de corrección
   - Riesgos

## Output
- `hotfix/HF-XXX/fix_description.md`

## Restricciones
- No implementar el fix.
