# Baseline Discovery Input

## Proyecto
Sistema de Facturación Completo (Full Stack)

## Contexto
El proyecto es un sistema de facturación existente que implementa una arquitectura cliente-servidor con separación clara entre frontend y backend. Está diseñado específicamente para el mercado argentino, integrando normativas fiscales locales (AFIP).

## Objetivo del Discovery
Realizar un relevamiento técnico-funcional exhaustivo del estado actual del proyecto (AS-IS) sin modificar su código, para establecer una base de gobierno sólida que permita la introducción de flujos de trabajo multi-agente.

## Alcance
- Análisis estático del código fuente.
- Inferencia de arquitectura y stacks tecnológicos.
- Mapeo de dominio de negocio (Facturación Argentina).
- Detección de deuda técnica y riesgos.
